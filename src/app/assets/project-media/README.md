# Project Media Assets

This folder contains local visual quick-look assets used by Story Mode and Recruiter Mode project cards.

## Files
- `blockopoly-board.svg`: Blockopoly multiplayer board preview.
- `curide-flow.svg`: CURide booking and trust flow preview.
- `crossing-cafe-menu.svg`: Crossing Cafe menu and ordering preview.
- `wishlist-gifting.svg`: Wishlist collaborative gifting dashboard preview.
- `deardays-calendar.svg`: Dear Days shared calendar preview.
- `storybot-pipeline.svg`: Storybot automation pipeline preview.
- `nba-shotmap.svg`: NBA shot trends analytics preview.

## Usage
Assets are imported in `src/constants/substance.ts` and attached to each project via the `media` field.
